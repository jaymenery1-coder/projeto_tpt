Este projeto é uma aplicação web interativa desenvolvida em Python com Streamlit. 
O objetivo é fornecer ferramentas visuais para explorar um conjunto de dados de anúncios de veículos.

## Funcionalidades
- Visualização de histograma do odômetro.
- Gráfico de dispersão entre preço e odômetro.
- Interface interativa via navegadores web.

## Como rodar localmente
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute o app: `streamlit run app.py`
